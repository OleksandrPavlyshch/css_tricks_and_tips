Size units

EM

Размеры в em – относительные, они определяются по текущему контексту.

1em – текущий размер шрифта.

		p {
			font-size: 20px;
			padding: .5em; //10px
			margin: 2em; //40px
		}
		.small {
			font-size: .8em; //16px
		}

Добавить демо!!!

REM

Единица rem задаёт размер относительно размера шрифта элемента.

		html {
			font-size: 16px;
		}
		p {
			font-size: 1rem //16px
			padding: .5eem // 8x
			margin: 2rem // 32px
		}

Добавить демо!!!

CH

ch задает размер относительно размера символа ноля "0"

		p {
			width: 35ch;
	// ширина 35 символов заданого размера шрифта
		}
