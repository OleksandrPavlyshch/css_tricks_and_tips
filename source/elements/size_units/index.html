Size units

EM

Размеры в em – относительные, они определяются по отношению к первому родителю которому задан размер шрифта.

		p {
			font-size: 20px;
			padding: .5em; //10px
			margin: 2em; //40px
		}
		.small {
			font-size: .8em; //16px
			margin: .1em; //20px
		}

REM

Единица rem задаёт размер относительно размера шрифта html элемента.

		html {
			font-size: 16px;
		}
		p {
			font-size: 1rem //16px
			padding: .5rem // 8x
			margin: 2rem // 32px
		}

CH

ch задает размер относительно размера символа ноля "0"

		p {
			width: 35ch;
	// ширина 35 символов заданого размера шрифта
		}

vmin, vmax

Vmin отсчитывается от размера меньшей стороны.

А Vmax от размера большей стороны.
